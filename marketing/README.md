# Builders Stream Marketing Pack

This folder contains ready-to-adapt marketing copy for Builders Stream.

- `positioning.md` — brand foundation, audience, messaging, and differentiators
- `landing-page-copy.md` — website-ready homepage copy
- `sales-one-pager.md` — concise customer-facing overview for a PDF or leave-behind
- `launch-messages.md` — email and social launch copy

The copy reflects the current MVP direction: company administration, CRM, project management, estimating, and e-signature approval. Accounting integrations and advanced compliance tooling are described as future capabilities, not current product promises.
